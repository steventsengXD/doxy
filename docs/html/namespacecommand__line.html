Detailed Description

Functions to parse command line arguments.

Module to read command line arguments to a program We assume they are of the form name=value. There must NOT be spaces around the '=' sign! Value can be extracted as a string, a long-integer or a double-precision real. Argument names are limited to 20 chars, and values to 30 chars as read.

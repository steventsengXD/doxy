Detailed Description

Read arguments by name or number.

All of the members of this interface take either a 'name' or a 'num' parameter, detailling which argument to look up and the type of the 'val' argument dictates the type to attempt parsing the argument as

Parameters
nameName to look up (if used)
numArg. number to look up (if used)
valValue to read into
existsWhether the name was found
Returns
True if the name is found and parsed, False otherwise
